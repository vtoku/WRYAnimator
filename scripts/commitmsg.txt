Discovered MotionBuilder 2026 installed locally and built a headless
verification harness (motionbuilder.exe -batch + scripts/mobuCheck.py,
target via WANIM_FBX env var). Bisected against MoBu's own saved output:

- Legacy warning fix: Creator strings "FBX SDK/FBX Plugins version 2020.3.7"
  (the SDK parses this form to date the file; even Blender's export warns),
  plus OtherFlags/TCDefinition 127, FBXHeaderVersion 1004, FBX version 7700.
- Mesh fix: Properties70 DefaultAttributeIndex=0 on mesh Models - without it
  the SDK template default (-1 = no active attribute) silently imports the
  mesh as a Null. Also added per-vertex normals LayerElement to the embedded
  face geometry (extracted from the GLB).
